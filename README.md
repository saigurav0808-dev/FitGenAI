FitGenAI is an AI-based personalized fitness planner that generates
workout and diet plans based on user inputs such as age, gender,
fitness goal, and activity level.

## Features
- Personalized workout plan
- Personalized diet plan (Indian food based)
- Simple and user-friendly interface
- Mobile responsive design
- Dataset-based recommendations

## Tech Stack
- Python
- Flask
- Pandas
- HTML, CSS
- CSV Dataset

## Dataset
The project uses an Indian food nutrition dataset to recommend
diet plans based on calories and fitness goals.

## How to Run
1. Install Python
2. Install required libraries:

## pip install flask pandas
3. Run the application:

## python app.py
4. Open browser and go to:
http://127.0.0.1:5000/


## Project Use Case
This project helps users generate a basic fitness and diet plan
without requiring a personal trainer.
